== Description ==

Recipe block is a custom Gutenberg Block That has the following  upgradation to be followed they are 
1) User can select multiple image's for the Recipe slider Image.
2) User can select the Recipe Type.
3) User can enter Prepration Time,Cook Time, Adddtional Time & generate Total times from the PREP,COOK & Adddtional Time.
4) User can also change the background color,  tittle color and text color 
5) User Can Add Ingredients to the Recipe,Nutrition Facts, Recipe Steps.
6) User can Add the Recipe Title Name.
7) User can add the Border Color,Border radious,Border Style,Block Padding,Title color, Sub-title color, Sub-Content Color,Block Background Color.

FEATURES
Responsive
Recipe Slider Image
Background color
Title text color
Description text color

== Installation ==

TO INSTALL THE PLUGIN AUTOMATICALLY:
Through WordPress admin, use the menu: Plugin > Add New
Search for Slider Block
Click on install then click the activation link
